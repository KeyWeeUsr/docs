=====================
Code Pattern Language
=====================

To specify code patterns, we use a language called `Cody`. The main goals of this language is
to be easy to read, write and understand, but powerful enough to describe any imaginable code
pattern in a straightforward way.

To get started, have a look at the :doc:`basic tutorial <basics>` first. To get an overview of
all available language constructs and operators, check ou the :doc:`operators <operators>` section.
